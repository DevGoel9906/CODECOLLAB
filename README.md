# CodeCollab 

**A curated open-source collaboration platform for students.**

CodeCollab is a premium web-based platform designed to help students discover, join, and collaborate on open-source projects. It simplifies the transition from learning to contributing by acting as a discovery and collaboration layer on top of GitHub.

---

## ✨ Features

- 🔍 **Project Discovery:** Browse projects by tech stack (React, Python, AI, Security, etc.) with real-time filtering.
- 🤝 **Structured Join System:** Apply to join projects and track your request status (Pending/Accepted/Rejected).
- 🛠️ **Project Management:** Project owners can review requests, manage contributors, and promote users to admins.
- 📊 **User Dashboard:** Track your contributions, requests, and project progress in one place.
- 👤 **Profile Customization:** Personalize your profile with a bio, avatar, and skill tags.
- 🔗 **GitHub Integration:** Direct links to repositories and automatic fetching of repository details.
- 🎨 **Premium UI/UX:** Dark-themed, glassmorphic design with smooth animations and a responsive layout.

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, Vanilla CSS3, Modern JavaScript (ES6).
- **Design:** Glassmorphism, Google Fonts (Outfit), Custom CSS Animations.
- **Persistence:** Browser `localStorage` for dynamic data (requests, profiles, user projects).
- **Organization:** Modular file structure with centralized data management.

---

## 📂 Project Structure

- `/all_projects`: Detailed project templates and view logic.
- `/admin_and_user_settings`: Management portals and settings pages.
- `/user_dashboard_system`: Activity tracking and user stats.
- `/data`: Centralized database (`project_database_file.js`) and backups.
- `code collab main page.html`: The central entry point for the platform.

---

## 🚀 Getting Started

Since this platform is designed as a direct-execution web app, you can get started immediately without complex set-up:

1.  Clone the repository:
    ```bash
    git clone https://github.com/404Dev-notFound/CODECOLLAB.git
    ```
2.  Open `code collab main page.html` in any modern web browser.
3.  Start exploring projects!
